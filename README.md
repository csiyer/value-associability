# Value Associability Project

This repo contains the following:
- `associability-task` : a task displaying cards with values, and then directly testing for value memory to measure image-level value associability.
- `episodic-choice-task` : a simplified version of the task from [Duncan and Shohamy 2016](http://duncanlab.org/wp-content/uploads/2018/04/Duncan_2016.pdf) to test the impact of value associability on episodic choice behavior
- `pilot/jonathans-data`: reanalysis of data from [Nicholas et al. 2022](https://elifesciences.org/articles/81679), a version of the incremental-episodic task, to assess (1) how image memorability affects episodic choices, and (2) how reliable image-level episodic choices are.
- `pilot/pilot-mem-choice`: task code and pilot data for a task asking if simply pairing memorable images with high values biases choices and inferences of deck value.
- `pilot/pilot-mem-inc-ep`: task code and pilot data for an incremental-episodic task using (1) high- and low-memorability object images, and (2) distinctive vs. non-distinctive objects to modulate episodic memory strength.
- `pilot/stimuli` contains the stimuli used in these pilot experiments.
- `stimuli` contains stimuli sampled from the THINGS dataset with high, medium, and low image memorability. Rebuild this folder with `selectstimuli.py`.
