# Re-analysis of Jonathan's data

From [Nicholas et al. (2022)](https://elifesciences.org/articles/81679).

We analyzed
1) Does image memorability (computed with [MemNet](http://memorability.csail.mit.edu/download.html)) predict episodic choice in the incremental episodic task? (answer: no)
2) Does image memorability predict subsequent memory when directly tested on the values of images? (answer: no)
3) Do certain images show reliable patterns of episodic choice across people? (answer: yes, sort of)
4) Do certain images show reliable patterns of direct value memory across people? (answer: yes, sort of)
